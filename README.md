# Project setup
Launch project<br>
`npm install`

Launch server<br>
`npm run dev`

Launch build //create *dist* folder - project version to deploy<br>
`npm run build`

Deploy app<br>
`"devDependencies": {
     "@vue/cli-plugin-pwa": "^4.5.0", 
}`
